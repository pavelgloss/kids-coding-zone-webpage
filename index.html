How I Teach?

  • Real programming language - no Scratch, but simplified JavaScript-based framework
  • No need to type everything manually - use prepared text blocks
  • Emphasis on trial and error - write code and immediately see results
  • Interactive platform similar to Scratch, but with actual coding
  • Learn critical thinking and computer science skills essential for the modern world

Example of our code:


backdropColor("pink")
image("puppy")

apple = image("apple")
apple.rotate(45)
apple.move(100)

This code is immediately interpreted and visually rendered!
